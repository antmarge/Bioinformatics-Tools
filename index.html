    countTA.pl

    A perl script that takes a fasta file and counts the number TA sites. This is useful for Transposon Sequencing when we want to know the ratio of transposon insertions (which happen at TA sites) to TA sites for a given region. For example, the ratio is used in the essentails analysis which calculates the p-value for a region given insertion representation

    getCoordsGBK.py

    A python script that takes a genbank file and outputs a file with all the gene ids and their genomic start and end coordinates. This script is useful for creating genomic tracks in Gviz (R package).
